**ABOUT BrenmaxOS:**
 - 32 bit
 - ring 0
 - minimal gui
 - no file system
 - legacy boot only

**ABOUT PROJECT:**
This project was started on the 11th of April 2026 as a kind of beginner project. I am still relativeley new to OSDev. The reason I made a GUI is because I just for some reason really like GUIs.

**WHY MAKE ONE:**
This operating system is made for fun. Not for school or anything.
